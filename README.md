### :open_file_folder: __Директории:__
| Папка | Описание |
|:------|:---------|
| ***`/components`*** | для Реакт-компонентов |
| ***`/images`*** | для изображений проекта |
| ***`/vendor`*** | для файлов сторонних разработчиков |
| ***`/utils`*** | для вспомогательных функций/запросов к API |
